# BP_algorithm

This respository completes the first task of the course **Machine Learning** , which requires us students tackle the XOR problem, including two input features and one output labels. The task is building a neural network with one hidden layer via BP algorithm.

## About the repot
You just need to open `data/main.pdf`, and you can see the report. To see more, please enter `data/main.tex` to vie the latex code.

## Running Procedure
First install the required packages
``` bash
pip install -r requirements.txt
```
Then run the code
```bash
python main.py
```
